O que os alunos devem observar

  • O HTML deve usar elementos semânticos como header, main, section, article e form.
  • O CSS deve organizar a tela com cards, grid, espaçamento, cores leves e estados visuais.
  • O JavaScript escuta eventos de formulário, valida dados simples e usa localStorage.
